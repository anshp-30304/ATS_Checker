Resume Keyword Matcher

The Resume Keyword Matcher helps job seekers analyze their resume and match it against job descriptions to identify relevant keywords. It can help improve the resume by focusing on keywords that are important to the employer.

Features

  • Upload your resume and job description.
  • Keyword matching algorithm to identify relevant keywords.
  • Visual progress indicators with a circular progress bar.
  • Interactive results showing matching keywords.
  • Responsive design for all screen sizes.

Technologies Used

  • React
  • HTML, CSS
  • JavaScript
  • React Circular Progressbar
